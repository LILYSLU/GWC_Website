About Us

Welcome to the GBN Girls Who Code Website. Some fun things we do here include: speaker meetings with women in tech, projects, culture of tech discussions, sisterhood bonding and more! GWC is a community for female and non-binary students to learn about coding and technology in a welcoming space. Check out the meeting schedule to view our meeting times :)

What is Computer Science?

Computer science is the study of all of the different ways computers can be used to make things easier, faster, or more fun. Computer science is used in a ton of fields -- from medicine, to art, to robotics, and media (think the news, Youtube, Tik Tok, and Netflix)! Coding is writing instructions for a computer in a language it can understand. Coding is just one small part of computer science. In your Club, you're going to be doing more than writing code -- you're going to practice thinking and creating like a computer scientist. Computer scientists are people who study all the different ways that code and computers can be used to create new technology. Just like a computer scientist, you'll try on a bunch of different hats, like Designer, Problem Solver, and Coder!
